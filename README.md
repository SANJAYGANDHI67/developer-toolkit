# 🛠️ DevKit - Developer Toolkit

A modern all-in-one Developer Toolkit built using HTML, CSS, and JavaScript.

DevKit provides essential tools that developers use daily, all in a single fast, responsive, and browser-based application. Everything runs locally in your browser with no server required.

## 🚀 Live Demo

https://sanjaygandhi67.github.io/developer-toolkit/

---

## ✨ Features

### 🔐 Password Generator
Generate secure and customizable passwords with support for:
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters
- Adjustable length

### 📱 QR Code Generator
Create QR codes instantly from:
- URLs
- Text
- Contact information

### 📄 JSON Formatter
- Beautify JSON
- Minify JSON
- Validate JSON structure

### 🔄 Base64 Encoder / Decoder
- Encode text to Base64
- Decode Base64 back to text

### 🎨 Color Palette Generator
- Generate random color palettes
- Copy HEX codes instantly
- Useful for UI/UX designers

### 🆔 UUID Generator
- Generate RFC-4122 compliant UUIDs
- One-click copy support

### 📝 Text Utilities
- Word Counter
- Character Counter
- Uppercase Converter
- Lowercase Converter
- Text Formatting Tools

### 📖 Markdown Previewer
- Write Markdown
- Live Preview
- Developer-friendly editing experience

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/sanjaygandhi67/developer-toolkit.git
```

Navigate to project folder:

```bash
cd developer-toolkit
```

Open:

```bash
index.html
```

in your browser.
